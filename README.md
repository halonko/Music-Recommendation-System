# Music-Recommendation-System

#Prerequirements:
  - Python 3.6
  - pandas 0.25.3
  - scikit-learn 0.22.1
  - seaborn 0.9.0
  - spotipy 2.6.1
  - pylab

# Installing:
  - Clone project
  - Install requirements.txt
  - Get spotipy user_id and user_secret
  - Create credentials.py in folder code
  - Copy to credentials.py
    user_id = XXX
    user_key = XXX
  - Run playlist_recommendation.ipynb
